# SimpleFlowLayout

简单的流式布局，继承自ViewGroup，方便自己扩展
原博客网站：[SimpleFlowLayout](https://blog.csdn.net/sj617913246/article/details/84136177)  
使用方法就和普通的ViewGroup一样，可自行扩展其他功能：
比如长按删除，单机选中，view动画效果等等
具体使用方法，请下载Demo查看
